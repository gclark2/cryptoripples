Notes:

-> git: track changes to files
	commands:
		git add <insert filename here> - add filename to repository
		git commit -m "<insert message here>" - commit changes to repo, message is a description of the changes
		git push - push changes to the internet repo

-> heroku: interface with the heroku app (how you make heroku computers run your code for you)
	commands:
		git push heroku master - push app to heroku
		
		heroku open - open app on heroku
